dB、dBm与dBw

dB

​ dB是无线通信中的基本概念。如“传播损耗是 dB”、“发射功率是dBm”、“天线增益是dBi”。

​ 最常见的3dB，它在功率图或者误码率图中经常出现，下降3dB就是指功率下降一半，3dB点指的就是半功率点。

​ +3dB表示增大为原来的两倍，-3dB表示下降为原来的1/2.

dB = 10lg(P1/P0)

​ dB表示功率P1相对于参考功率P0的大小关系。

​ 常用的dB值中，+10dB表示功率增加为原来的10倍，-10dB表示功率减小为原来的1/10.

​ 可见，dB是一个相对值，它的作用是把一个很大或者很小的数，用一个简短的形式表达出来。这可以极大的方便我们的计算和描述。尤其是在绘制表格的时候，如果没有换算成dB，那么坐标轴将很长无法表示。

功率变化增大到原来的100000000倍，用dB表示为10lg10^8 = 80dB，减小到原来的0.00000001倍，用dB表示为-80dB。

dBm与dBw

​ 这是最常用的两个单位。它们对应的就是把dB公式中的参考功率P0分别变换成1mW、1W。由于1mW、1W都是确定的值，因此dBm和dBw都可以表示功率的绝对值。

​ 这里需要特别记住的是1W=30dBm。

例子：44dBm = 30dBm + 10dB + 10dB - 3dB -3dB，
因此44dBm = 1W×10×10/2/2 = 25W

​ 这里需要注意，等式右侧除了30dBm，其余的拆分项都要用dB表示。即用一个dBx减另一个dBx时，得到的结果用dB表示。如功率46dBm比功率40dBm大6dB。

​ 另外要注意dB是相对值，而dBm和dBw则对应的是绝对值。

例如46dB表示P1为P0的4万倍，46dBm表示P1的值为40W。

​ dB家族中常见的还有dBi、dBd、dBc。 它们的计算方法与dB的计算方法完全一样，表示的还是功率的相对值。不同的是，它们的参考基准不同，即分母上的参考功率P0 所代表的含义不同。

​ 此外，dB家族不仅可以表示功率的增益和损耗，还可以表示电压、电流、音频等，大家要具体场景具体应用。需要注意的是，对于功率的增益，我们用10lg(Po /Pi )，对于电压和电流的增益，要用20lg(Vo /Vi )、20lg(Io /Ii)。
